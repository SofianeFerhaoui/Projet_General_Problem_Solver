# Projet_General_Problem_Solver
##  Description
Ce projet est une implémentation en langage C de l'algorithme General Problem Solver . Il s'agit d'un programme d'intelligence artificielle classique conçu pour résoudre des problèmes complexes.

## Contenu
* program.c : code source principal du moteur de résolution.
* RAPPORT_General_Solver.pdf : Rapport complet détaillant la conception et les tests.
* monkeys.txt, school.txt, blocs.txt : Différents scénarios de tests logiques fournis.

## Utilisation
Pour tester (vscode):
1. Compilez avec GCC : gcc program.c -o program
2. executez avec : ./program

## Objectif Académique
Ce projet démontre ma capacité à structurer un algorithme complexe, à manipuler des structures de données en C et à appliquer des concepts théoriques d'IA à des problèmes concrets.
